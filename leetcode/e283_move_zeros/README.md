## 283. Move Zeroes

### 题目描述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**注意**：必须在原数组上操作，不能拷贝额外的数组。尽量减少操作次数。

### 示例

```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

### 解题思路

#### 方法1: 双指针法（两次遍历）

1. 第一次遍历：使用快慢指针，将所有非零元素移到前面
2. 第二次遍历：将剩余位置填充为0

**时间复杂度**: O(n)  
**空间复杂度**: O(1)

#### 方法2: 交换法（一次遍历）

使用双指针，当遇到非零元素时，与慢指针位置交换。

**时间复杂度**: O(n)  
**空间复杂度**: O(1)
