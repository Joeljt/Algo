## 字典

字典是一种高级数据结构，用于存储键值对，可以基于很多不同的数据结构实现。

这里我们选择使用链表、二叉搜索树、AVL树、红黑树以及哈希表来实现字典。

- [链表](./list/README.md)
- [二叉搜索树](./bst/README.md)
- [AVL树](./avl/README.md)
- [红黑树](./rbt/README.md)
- [哈希表](./hash/README.md)

同时，由于集合与字典的实现非常相似，本着组合大于继承的原则，这里我们在字典的基础上进行简单的封装即可实现集合。

由于字典本身的 key 不可重复，我们可以忽略 value 的值，直接使用 key 来表示集合中的元素。Set的"不重复"特性可以由Map的"key不重复"特性自然保证。

- [集合](./set/README.md)
